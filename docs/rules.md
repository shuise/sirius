# 规范

## 开发规范
1. 开发框架 https://plasmo.happyfe.com/framework
2. 技术栈 react + typescript + tailwindcss
3. 每个具体功能的实现都单独一个文件，文件名就是功能的名称，每个文件的代码都要符合 typescript + tailwindcss 规范
4. 公共通用文件放入 common


## 产品说明
1. 强制侧边栏模式
2. 所有插件 ui 都放入侧边栏
3. 侧边栏 3 个菜单：功能、系统设置、产品说明
4. 已有功能：见 README.md

## AI 相关
1. 实现通用 ai 对接服务
2. 书籍提取及识别通过 ai 处理，并汇总列表到侧边栏
