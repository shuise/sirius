# 功能修改：

1. 插件改为点击后生效，默认不注入
2. 功能设置和系统设置合并为一个页面
3. 增加一个页面「讨论」，正文中选择一段话，右键直接发送到讨论页面，AI 回答
4. 在「讨论」页面的发送按钮右侧增加一个「导出会话」按钮，点击后讲当前会话导出为 md

## 摘要部分的调整思路
1. 启动插件后，先提取摘要和最重要的书籍列表（不超过 15 个）,通过控制台打印出来
2. 摘要直接展示
3. 书籍通过微信读书的接口查询，查不到的过滤掉，查到的再展示。

## 会话管理
1. 每个页面一个会话，在独立的会话中讨论
2. 所有的数据检索共用同一个会话
3. 通过插件发送的讨论在「讨论区」展示重复。
